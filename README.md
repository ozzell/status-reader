# Status Reader

## Prerequisites

* Node 18

## Getting Started

```bash
# To install dependencies
npm install

# To run the app
npm start
```

Open the dev environment at [http://localhost:8080](http://localhost:8080)

## Other commands

Running tests

```bash
npm run test
```

Running linter

```bash
npm run lint
```

Building the application

```bash
npm run build
```
